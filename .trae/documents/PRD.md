
# 小红书运营周报看板 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
本项目旨在创建一个小红书运营周报看板网页，帮助内容创作者和运营人员可视化分析小红书账号的运营数据。通过上传数据表格，系统自动进行数据分析并生成可视化报表。

### 1.2 产品定位
- **目标用户**：小红书内容创作者、运营人员、数据分析人员
- **核心价值**：提供直观的数据可视化，帮助用户快速了解账号运营情况，发现爆款内容规律，获取运营建议

### 1.3 设计风格
- 现代简约的数据看板风格
- 渐变色统计卡片，视觉吸引力强
- 清晰的信息层级，数据一目了然
- 配色方案：白背景配合多彩渐变卡片，专业且活泼

## 2. 功能需求

### 2.1 核心功能

| 功能模块 | 功能描述 | 优先级 |
| :--- | :--- | :--- |
| 数据上传 | 支持上传Excel/CSV数据表格 | 高 |
| 数据统计卡片 | 展示总曝光量、总观看量、总互动数、新增粉丝、总粉丝数 | 高 |
| 内容类型对比 | 对比视频笔记和图文笔记的表现数据 | 高 |
| 数据洞察 | 自动分析内容类型表现差异，给出建议 | 高 |
| 爆款笔记排行 | 按曝光量排序展示热门笔记 | 高 |
| 详细数据表格 | 展示所有笔记的详细数据 | 高 |
| 运营建议 | 基于数据分析提供优化建议 | 高 |

### 2.2 数据字段定义

| 字段名 | 说明 | 数据类型 |
| :--- | :--- | :--- |
| 笔记标题 | 小红书笔记的标题 | 字符串 |
| 体裁 | 内容类型：视频/图文 | 字符串 |
| 曝光 | 笔记曝光量 | 数字 |
| 观看 | 笔记观看量 | 数字 |
| 点击率 | 观看量/曝光量 | 百分比 |
| 点赞 | 点赞数量 | 数字 |
| 评论 | 评论数量 | 数字 |
| 收藏 | 收藏数量 | 数字 |
| 分享 | 分享数量 | 数字 |
| 发布日期 | 笔记发布日期 | 日期 |

### 2.3 数据分析逻辑

#### 2.3.1 内容类型对比分析
- 统计视频笔记和图文笔记的总曝光量、观看量、点赞数、评论数
- 计算各类型占比，判断哪种内容表现更好
- 生成数据洞察建议

#### 2.3.2 爆款笔记判定
- 按曝光量降序排列
- 标记曝光量最高的笔记为爆款
- 展示爆款笔记的互动数据

#### 2.3.3 运营建议生成
- 基于内容表现给出优化方向
- 根据爆款笔记特征提炼成功经验

## 4. 技术需求

### 4.1 技术栈
- **框架**: Vue 3 + TypeScript
- **UI组件**: Element Plus
- **图表库**: Chart.js 或 ECharts
- **样式**: Tailwind CSS 3
- **Excel解析**: xlsx 库

### 4.2 数据格式要求
- 支持 CSV 和 Excel (xlsx) 格式
- 数据编码：UTF-8
- 必须包含的列：笔记标题、体裁、曝光、观看、点赞、评论、收藏、分享

---

## 5. 非功能需求

### 5.1 性能要求
- 页面加载时间 < 2s
- 数据解析响应时间 < 1s（100条数据以内）

### 5.2 兼容性要求
- 支持主流浏览器：Chrome、Firefox、Safari、Edge
- 响应式设计，支持桌面端和移动端

### 5.3 易用性要求
- 数据上传操作简单直观
- 图表和数据展示清晰易读
- 支持数据导出功能
